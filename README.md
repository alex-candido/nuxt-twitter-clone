- Node version v18.15.0

- Clone this repository

```
$ yarn add
```

- Setup .env file
```

NEXTAUTH_JWT_SECRET=

```
- Start Prisma
```
$ yarn prisma generate

$ yarn prisma db push
```

- Start the project
```
$ yarn dev
```
